## DatasetSpec

Dataclass defining the static properties of a dataset.

<ParamField path="name" type="str" required>
  Human-readable dataset name
</ParamField>

<ParamField path="version" type="str" required>
  Dataset version identifier
</ParamField>

<ParamField path="train_path" type="str" required>
  Relative or absolute path to training CSV file
</ParamField>

<ParamField path="test_path" type="str" required>
  Relative or absolute path to test CSV file
</ParamField>

<ParamField path="expected_features" type="int" default="784">
  Number of feature columns (excluding label)
</ParamField>

<ParamField path="expected_min_rows" type="int" default="100">
  Minimum rows required for validation
</ParamField>

<ParamField path="download_base_url" type="str" default="https://pjreddie.com/media/files">
  Base URL for downloading dataset files
</ParamField>

## validate_dataset_file()

Validate dataset integrity at both file and tensor levels.

```python
validate_dataset_file(
    path: str | Path,
    expected_features: int,
    expected_min_rows: int,
    expected_sha256: Optional[str] = None
) -> Tuple[int, int]
```

<ParamField path="path" type="str | Path" required>
  Path to dataset CSV file
</ParamField>

<ParamField path="expected_features" type="int" required>
  Expected number of feature columns (excluding label column)
</ParamField>

<ParamField path="expected_min_rows" type="int" required>
  Minimum number of rows required
</ParamField>

<ParamField path="expected_sha256" type="Optional[str]" default="None">
  Optional SHA256 hash for file integrity verification
</ParamField>

**Returns:** Tuple of `(n_rows, n_cols)` for the validated dataset

**Raises:**
- `FileNotFoundError` if file does not exist
- `ValueError` if file is empty, hash mismatch, shape mismatch, too few rows, contains NaN, or labels out of range [0,9]

## ensure_dataset_ready()

Validate dataset availability and optionally auto-download if missing or invalid.

```python
ensure_dataset_ready(
    spec: DatasetSpec,
    expected_features: int,
    expected_min_rows: int,
    auto_download: bool = False,
    expected_sha256: Optional[str] = None
) -> Tuple[int, int]
```

<ParamField path="spec" type="DatasetSpec" required>
  Dataset specification
</ParamField>

<ParamField path="expected_features" type="int" required>
  Expected feature count
</ParamField>

<ParamField path="expected_min_rows" type="int" required>
  Minimum row count
</ParamField>

<ParamField path="auto_download" type="bool" default="false">
  If True, automatically download dataset when validation fails
</ParamField>

<ParamField path="expected_sha256" type="Optional[str]" default="None">
  Optional SHA256 for integrity check
</ParamField>

**Returns:** Tuple of `(n_rows, n_cols)` after successful validation

**Raises:**
- Validation errors if `auto_download=False` and dataset is invalid
- `RuntimeError` if auto-download fails
